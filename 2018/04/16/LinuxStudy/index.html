在装虚拟机中安装 CentOS7 并配置

发表于 2018-04-16 | 分类于 Linux

准备工作

  1. 下载 CentOS 镜像文件，官网： https://www.centos.org
  2. 下载并安装 VMware
  3. 创建如下目录，mv 用于存放虚拟机文件，dk 用于存放虚拟机磁盘文件

创建虚拟机

打开 VMware 软件，新建虚拟机，并下一步，没截图的地方，默认配置，下一步即可

选择 CentOS7 镜像文件

设置虚拟机名称，选择虚拟机存放位置，刚创建的 vm 目录

选择 NAT 模式

选择将磁盘拆分成多个文件，这样单个文件较小，方便管理

设置磁盘文件目录，刚创建的 dk 目录

勾选上创建后开启此虚拟机，并点击完成

可以用上下来移动光标，回车确认选择安装

再次回车确认

等待一会，出现如下界面，往下拉，选择中文即可

点击安装源

点击完成，默认设置即可

点击安装位置

点击完成，默认设置即可

开始安装

正在安装中，等待约 10 分钟

设置 ROOT 用户的密码，密码设置简单的话，需要点击两次完成

等待安装完成

安装完成，重启

配置虚拟机

输入 root 用户名，所设置的密码，即可进入，注意：密码输入不显示

看到次界面，就代表登陆成功

虚拟机名称右键，设置，网络适配器，确认选择 NAT 模式

编辑，虚拟机网络编辑器

选择 NAT 模式，分别查看 NAT 设置与 DHCP 设置

查看 NAT 设置，记录子网掩码与网关，一会会用到

查看 DHCP 设置，选择一个在起始，结束之间的 ip 地址并记录

在本机查看网络状态，如果是用网线就查看对应的状态

详细信息，记录 DNS 服务器地址

进入 linux 命令窗口，输入命令 vi /etc/sysconfig/network-scripts/ifcfg-ens33，即看到如下配置信息

修改 BOOTPROTO=static 代表 ip 设置为静态的，修改 ONBOOT=yes，代表系统启动就开启网络，并添加之前所记录的信息，具体如下（i 命令进入修改，修改完成后，按 esc，wq 保存并退出）

IPADDR=192.168.40.128
GATEWAY=192.168.40.2
NETMASK=255.255.255.0
DNS1=192.168.43.1


用clear命令清屏，输入 service network restart使配置文件生效，输入ifconfig命令，查看设置后的 ip 地址

分别 ping 本地电脑 ip 地址，与百度地址，即可查看是否配置成功
可以看到 www.baidu.com 只是一个别名，真实地址为 www.a.shifen.com

到此，就安装并配置好 CentOS7 虚拟机了
