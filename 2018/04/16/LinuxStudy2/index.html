Linux 下搭建 Java 环境

发表于 2018-04-16 | 分类于 Linux

上一篇文章演示了在虚拟机中安装 CentOS7，下面介绍如何在 Linux 下搭建 Java 环境

采用 SecureCRT 连接虚拟机，采用这款软件连接，可以粘贴复制命令，比较方便，前提虚拟机得开启

执行如下命令，进入 /opt 目录，创建并进入 java7 目录

cd /opt
mkdir java7
cd java7

采用 WinSCP 软件连接到虚拟机，这个软件可以上传本地文件到虚拟机里

上传 Linux 版的 jdk 到虚拟机 /opt/java7 目录上，并设置全部权限

执行 ll 命令，即可看到刚上传的 jdk 压缩包

执行如下命令，即可解压该压缩包到当前文件夹，再次 ll 命令，可以看到解压后的文件

tar -zvxf jdk-7u79-linux-x64.tar.gz

执行 vi ~/.bash_profile命令，进入配置环境变量文件

在其中，添加如下配置，其它的不要动

export JAVA_HOME=/opt/java7/jdk1.7.0_79
export CLASSPATH=$JAVA_HOME/jre/lib/rt.jar:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
export PATH=$PATH:$JAVA_HOME/bin


执行source ~/.bash_profile，使配置文件生效，执行java -version，查看 jdk 版本

到此，Linux 下就搭建好了 Java 环境

# Java # Linux
在装虚拟机中安装 CentOS7 并配置
