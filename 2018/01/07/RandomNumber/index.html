生成指定范围内不重复的随机数

发表于 2018-01-07 | 分类于 算法

原理简介

生成指定范围内的随机整数的在上一篇文章里介绍了，这篇文章，在其基础上添加不重复，方法如下

方法一

方法一采用了Set集合，通过Set的特性，也就是Set集合中只能存放不同元素的特性，
首先定一个Set集合，用循环生成随机数，放入集合，
如果放入成功就执行下一次循环，失败就代表这个数字已经出现在集合中了，i--,重新执行本次循环，直到循环结束
此方法的有点是简单易懂，也不需要什么算法，
缺点就是随机出的数字是有序下的随机数，HashSet虽然是随机存放，但随机数所存放的顺序是已经固定的

Random ra = new Random();
Set<Integer> si = new HashSet<>();// 创建set集合，存放随机数

for (int i = 0; i < num; i++) {
    int ranNum = ra.nextInt(max - min + 1) + min; // 获取开始与结束之间的随机数
    if (!si.add(ranNum)) // 把随机数存到集合中,如果添加失败，代表重复，i--，重新循环
        i--;
}

方法二

方法二是把随机数存放在数组里面，首先在外层for循环里生成一个随机数，
把当前随机数存放在数组里，在通过内层for循环来判当前随机数是否与之前的每一个数相等，
如果相等，i--，重新本次循环
我带大家走一边流程，参数分别传入 min=1,max=10,num=5

第一次循环 i=0, 内层循环 j<=i-1 不成立，不进行比较
第二次循环 i=1, 内层循环 j<=i-1, 成立，此时 i 为1，j为 0 用 arr[0] 与 arr[1] 进行比较，重复的话 i-- 重新循环，
第三次循环 i=2, 内层循环 j 的范围位 0~1, 即用 arr[2] 与 arr[0],arr[1]进行比较，
第四次循环 i=3, 内层循环 j 的范围位 0~2, 即下标位3的与下位位0~2的进行比较，
第五次循环 i=4, 内层循环 j 的范围位 0~3, 即下标位4的与下位位0~3的进行比较，
如果以上比较有重复的，i--，重新本次循环，这样就随机到5个不同的数。

Random ra = new Random();
int[] arr = new int[num]; // 存放随机数数的数组

for (int i = 0; i < num; i++) {
    int ranNum = ra.nextInt(max - min + 1) + min; // 获取开始与结束之间的随机数
    arr[i] = ranNum;
    for (int j = 0; j <= i - 1; j++) {
        if (arr[i] == arr[j]) { // 当前随机数与前一个相等，i--，重新随机
            i--;
            break;
        }
    }
}

方法三

方法三，先定义一个count来计算有效个数，有效个数小于所要生成数组的数量时成立，
定义一个flag标志，初始位true，
生成一个随机数，用这个随机数与数组中的每一个数进行比较，
如果相等，标志改为false，然后下一次循环，
如果不想等，标志默认位true，把当前随机出来的数，存放在数组里，并有效数字加1，
当有效数组不小于数组长度时，退出循环，这是，不同的随机数就存放到数组里，
此方法每次生成的随机数都会与数组里每一个进行比较，因此效率比方法二略低

Random ra = new Random();
int[] arr = new int[num]; // 存放随机数数的数组

int count = 0; // 计算有效个数

while (count < arr.length) {
    int ranNum = ra.nextInt(max - min + 1) + min; // 获取开始与结束之间的随机数
    boolean flag = true; // 定一个标志，用来判断是否重复
    for (int i = 0; i < arr.length; i++) {
        if (ranNum == arr[i]) {
            flag = false;
            break;
        }
    }
    if (flag) {
        arr[count] = ranNum;
        count++; // 有效个数加一
    }
}
