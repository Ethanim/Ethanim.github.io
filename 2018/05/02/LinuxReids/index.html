Linux 下安装 Reids 并配置

发表于 2018-05-02 | 分类于 Reids

首先通过 WinSCP 把 Reids 上传到服务器上

进入 redis 目录，执行如下代码

执行 make 会报错，因为缺少gcc-c++，执行如下命令安装gcc-c++

安装好后，执行 make MALLOC=libc，来编译
在执行 make install 把编译好的文件添加到执行目录里
这样就安装好了 redis，但我们需要修改配置文件

vi redis.conf

把 bind 127.0.0.1 改为 bind 0.0.0.0
默认只能本机访问，修改为 0.0.0.0，代表任意机器都能访问

daemonize no 改为 daemonize yes，改为 yes，代表可以后台执行

指定配置文件，启动 redisredis-server ./redis.conf
查看是否启动成功与端口ps -ef | grep redis

redis-cli 就可以进入到客户端了

但是没有密码就可以访问，vi redis.conf，进入配置文件，把 requirepass 前面的 # 去掉，并设置密码，如下
requirepass 123456

进入客户端，关闭 redis，在启动 redis

下次访问，就需要就需要用密码来登录了
auth 123456
